Daisy UI is a Tailwind extension that uses class names for common UI components.

## Guidelines
1. Prefer daisyUI class names to Tailwind utility classes unless you have a specific reason not to.
2. When creating a new UI component, check whether daisyUI already has a component for it.
3. Load only the relevant reference/component file(s) for the current task.

## References
- Installation: `references/installation.md`
